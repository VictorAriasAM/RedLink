# RedLink - Red Social Web

RedLink es una aplicación web inspirada en las redes sociales modernas, diseñada para permitir la interacción entre usuarios mediante publicaciones, historias, páginas y contenido multimedia.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## Funcionalidades

* Página principal estilo red social.
* Publicación de contenido.
* Historias (Stories).
* Sección de amigos.
* Páginas de usuario.
* Feed de publicaciones.
* Diseño responsivo e interfaz moderna.
* Integración de imágenes y contenido multimedia.
